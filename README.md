# Robô de Afiliados

Bot integrado com API para automação de mensagens e anúncios.

## Instalação local

1. Crie o ambiente:
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # macOS / Linux
   source venv/bin/activate
   pip install -r requirements.txt
   ```

2. Execute:
   ```bash
   python app.py
   ```
